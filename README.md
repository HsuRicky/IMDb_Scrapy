# IMDb_Scrapy
>  first practice program，之後如果有新的網站實作，有機會在放在這邊
  
# 概述  
### **簡介:** 
>  ![IMDb](imdb_mark.jpg)("https://www.imdb.com/")   
>  抓取 IMDb 網站中電影及影集資料。
      
### **使用方式:**  
>  使用者輸入 IMDb 網站的電影或影集網址，若為電影網址，季數和集數不須輸入並可以抓取該電影各式資訊；若為影集網址，得輸入季數及集數，但若想爬全部季數影片，則季數輸入"all"，若想爬取某季數的影片，則級數輸入"all"。
